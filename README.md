# Drishti - Automated Attendance System

Drishti is an AI-powered automated attendance system designed for rural schools. It uses face recognition and gait analysis to mark attendance without manual effort, ensuring accuracy, speed, and transparency.

## 🚀 Features

- Face Recognition using deep learning models
- Gait Recognition for identifying identical twins
- Works with masked or partially covered faces
- Fully automatic attendance marking
- Real-time parent notifications (SMS/Email)
- Offline-first system with local storage
- Secure data handling using encrypted embeddings
- Manual override and reporting system

## 🧠 Tech Stack

- Python
- OpenCV
- ResNet (Face Recognition)
- MediaPipe / OpenPose (Gait Recognition)
- SQLite / CSV (Database)
- Twilio API (SMS)
- SMTP (Email)

## ⚙️ How It Works

1. Students are enrolled with multi-angle face data
2. Camera captures students as they walk in
3. System identifies using face recognition
4. If needed, gait recognition is used as backup
5. Attendance is automatically recorded
6. Parents receive instant updates

## 💡 Problem Solved
